FAQs

When compiling my classes, I get an error stating that some leJOS classes can not be found.

Make sure that NXJ_HOME is set in the the leJOS NXJ section of the Preferences page of Eclipse and that you have converted the project to a leJOS project.

When trying to upload my main class to the brick, I get an error stating that the communication classes can not instantiated.

This is due to the fact that the native comm driver library can not be loaded. Make sure that the environment variable NXJ_HOME is set and that %NXJ_HOME%\bin resp. $NXJ_HOME/bin is included in your system class path.
