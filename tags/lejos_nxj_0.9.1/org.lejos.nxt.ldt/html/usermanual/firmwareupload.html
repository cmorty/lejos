Uploading the firmware

To run leJOS NXJ programs on the NXT brick, you need to upload the leJOS NXJ firmware. Note that this only has to be done once.

To upload the firmware, use the leJOS NXJ→Upload Firmware menu item in the Eclipse menu bar or click on the leJOS NXJ icon on the toolbar. Messages will be displayed in the Eclipse console. You will be asked, to identify which NXT brick you want to upload the firmware to. If all goes well, you NXT brick will reboot with the new firmware.
