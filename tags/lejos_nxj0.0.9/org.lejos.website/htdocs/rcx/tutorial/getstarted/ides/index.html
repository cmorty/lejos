Lesson: Development environments for leJOS

Presently, there's quite a bunch of development environments available that can be used and configured for developing RCX applications with leJOS:

  • LDT: an Eclipse 3 plugin by Jochen Hiller and Markus Heiden
  • BlueJ with the leJOS-specific extension BJMT
  • JCreator
  • Jurtle
  • Textpad; for configuring it for leJOS see Walt White's page
