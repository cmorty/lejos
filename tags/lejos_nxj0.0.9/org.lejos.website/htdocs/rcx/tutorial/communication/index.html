Trail: Communication

This trail discusses classes of the leJOS platform that are concerned with communcation, that is sending commands or data
  • from the LEGO® Tower to the RCX and vice versa
  • from an RCX to another
  • from the LEGO® remote control to the RCX.

Rcxcomm

delivers an overview on the various possibilities to communicate with the RCX.

Using the LEGO® remote control

introduces the josx.rcxcomm.remotecontrol package.
