Creating a leJOS NXJ project

To compile programs with the leJOS NXJ API and upload them to the NXT brick, you need to create a project of the appropriate type. This is done the following way:
  1. Create (or import) a Java project.
  2. Select the project in the Package Explorer of the Java perspective and open its context menu.
  3. Navigate to the leJOS NXJ menu and select Convert to leJOS NXJ project. The project's nature will be changed and the classpath will be adapted so that the leJOS NXJ jars will be used for compilation and linkage.
Note; if some error occurs, consult Eclipse's Console view for details.
