Uploading a leJOS NXJ program to the NXT brick

To link and upload a leJOS NXJ program to the NXT brick, perform the following steps:
  1. In the Package Explorer of the Java perspective, select the .java file that contains the program's main class and open its context menu.
  2. Navigate to the leJOS NXJ menu and select Upload program to the NXT brick. The required classes will be linked and uploaded to the NXT brick.
  3. Use the leJOS menu on the brick to run the program.
Note; if some error occurs, consult Eclipse's Error Log view for details.
