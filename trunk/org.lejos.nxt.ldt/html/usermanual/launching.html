Uploading a leJOS NXJ program to the NXT brick

Note that LeJOS PC projects are launched the normal way, just like any other Java project.

To link and upload a program to the NXT brick, first make sure that the project is a LeJOS NXT project by checking for a little LeJOS icon next to the project name. If the project is not a LeJOS NXT project, or you haven't created a project yet, follow these instructions. Once a LeJOS NXT project is set up, follow these steps:

  1. In the Package Explorer of the Java perspective, right click on the project, a package, a *.java file or a class
  2. Select Run As→LeJOS NXT Program
  3. Eclipse might ask you to select the main class, if multiple suitable classes exist in your selection
  4. Two new files with the name of the class will appear in the root directory of the project
  5. Note that the verbosity of this feature (including logging of the signatures of the classes that get uploaded to the brick) can be enhanced by checking the verbose box of the leJOS NXJ preferences page.
  6. Use the leJOS menu on the brick to run the program.

Note; if some error occurs, consult Eclipse's Console view for details.
