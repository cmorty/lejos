Uploading the firmware

To run leJOS NXJ programs on the NXT brick, you need to upload the firmware that comes with the LEJOS NXJ to it.
Note: This has to be done only once.

  1. To upload the firmware, first set your NXT brick to reset mode by pressing the reset button located in one of the plug holes on the bottom side of the brick. Once the brick is in reset mode, it will start clicking in a low pulse.
  2. Now use the leJOS NXJ->Upload Firmware feature located in the menu bar of the Eclipse workbench to upload the firmware to the NXT brick. Once the upload is complete, an according message is displayed; the brick's LCD will show the leJOS logo. The brick is ready now to receive and run leJOS NXJ programs.
Note; if some error occurs, consult Eclipse's Error Log view for details.
