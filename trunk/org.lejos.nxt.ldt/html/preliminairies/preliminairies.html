Preliminaries

To use the Eclipse plugin for leJOS NXJ, you need to have the following requirements fulfilled:
  1. A distribution of leJOS NXJ installed on your local machine
  2. An environment variable NXJ_HOME pointing to the root directory of this installation
  3. %NXJ_HOME%\bin (for Windows) or $NXJ_HOME/bin (for Linux/Mac OS X) must be included in your system path
  4. If you want to use Bluetooth for communication of your computer with the brick, you need a working Bluetooth stack.
For requirements of leJOS NXJ itself, consult the README.html file contained in the distribution.
