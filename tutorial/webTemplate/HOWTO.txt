a HTML document has 2 main sections: head and body.
In head section is the part of any web document where you add:
+ metadata about your document
+ information about dependences:
- CSS files
- Icon
- Javascript

In body section is the part of any web document where you add:
- The contents

With OpenOffice, you discovered that the output is better because the HTML is more clearer than MS HTML.

Steps to integrate contents using HTML from Openoffice with leJOS  tutorial Web Template:

+ Make a back from your contents
+ choose your first folder, use the most easy and short document to experiment
+ rename the current document: contentX.htm -> contentXOld.htm, for example
+ paste into this folder the web template and rename the file with the name: contentX.htm
+ Open the old web document contentXOld.htm with your favorite webBrowser. I recommend firefox by security reasons.
+ Open the document contentXOld.htm with a text editor. I recommend Notepad++
+ Open the document contentX.htm with a text editor.
+ Find head section in contentXOld.htm and copy all metadata and all <style> tag
+ Paste this text into the end of head section in contentX.htm
+ Find the navigation system from contentXOld.htm (The links in the left side of the old web document) and copy the HTML
+ Find a HTML comment STEP 2: CUT & PASTE LEFT NAVIGATION SYSTEM FROM OPEN OFFICE and paste into this area
+ Make a HTML comment with final navigation system. ¿How to do a HTML comment? type in your text editor <!-- TEXT  TO COMMENT -->
